# SRIFOTON Machine Learning Competition 2025 by Sriwijaya University


# Overview
Kompetisi ini bertujuan mengembangkan model Machine Learning yang akurat dalam memprediksi dan mendeteksi penyakit pernapasan melalui citra X-Ray paru-paru pasien. Para peserta dalam melakukan pembuatan model, peserta diperkenankan untuk menggunakan berbagai metode, tools, library, atau framework apa saja.
